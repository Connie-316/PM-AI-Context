## 核心规范速查

### 单位换算

```
px × 2 = rpx
```

| 元素 | 设计稿(px) | 小程序(rpx) |
|------|-----------|-------------|
| 状态栏 | 22 | 44 |
| 导航栏 | 44 | 88 |
| TabBar | 50 | 100 |
| 页面边距 | 16 | 32 |
| 页面标题 | 18 | 36 |
| 正文 | 14 | 28 |
| 辅助文字 | 12 | 24 |

### 颜色规范

```css
/* 品牌色 */
--color-primary: #3370FF;      /* 主色蓝 */
--color-primary-dark: #2860E0;  /* 深蓝 */
--color-primary-light: #E8F0FF; /* 浅蓝（背景） */

/* 价格/行动色 */
--color-price: #F02C2C;         /* 价格红 */
--color-accent: #FF6B35;        /* 促销橙 */

/* 中性色 */
--color-title: #333333;          /* 标题 */
--color-body: #666666;          /* 正文 */
--color-aux: #999999;           /* 辅助 */
--color-line: #E5E5E5;         /* 分割线 */
--color-bg: #F5F5F5;           /* 背景灰 */
```

### 小程序限制

| 特性 | 支持 | 说明 |
|------|------|------|
| flex布局 | ✅ | - |
| border-radius | ✅ | - |
| box-shadow | ✅ | 性能消耗大，少用 |
| ::before/::after | ❌ | 伪类不支持 |
| CSS变量 | ❌ | 使用固定值 |

### 安全区域

```css
padding-bottom: env(safe-area-inset-bottom);
```
