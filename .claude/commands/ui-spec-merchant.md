# 商家端 UI 规范 Skill（ui-spec-merchant）

## 使用场景
当用户提到"商家端UI"、"商家后台UI"、"蓝色主题UI"、"管理后台UI"时调用。

---

## 一、技术栈

| 技术 | 说明 |
|------|------|
| 框架 | React + Ant Design 5 |
| 构建 | Vite |
| 主题色 | #1664FF（蓝色） |
| 辅助色 | 橙色 #FF7700→#FF6B00（统计卡）、价格红 #F5222D |
| 字体 | 微软雅黑 / -apple-system |

---

## 二、颜色规范

### 主色系（蓝色）

```css
--color-primary: #1664FF;
--color-primary-hover: #1458DB;

.ant-btn-primary { background: #1664FF !important; border-color: #1664FF !important; }
.ant-btn-primary:hover { background: #1458DB !important; }
.ant-btn-link { color: #1664FF !important; }
```

### 中性色

| 色值 | 用途 |
|------|------|
| #1A1A1A | 顶部导航/侧边菜单背景 |
| #FFFFFF | 卡片/表格背景 |
| #F5F5F5 | 页面背景 |
| #FAFAFA | 表格表头 |
| #333333 | 标题/正文 |
| #595959 | 正文/辅助 |
| #8C8C8C | 辅助灰 |
| #D9D9D9 | 边框 |
| #CCCCCC | 未选中菜单 |

### 功能色

| 色值 | 用途 |
|------|------|
| #F5222D | 价格红/危险操作 |
| #FAAD14 | 警告/待处理 |
| #52C41A | 成功/已完成 |
| #FF7700 | 橙色统计卡（渐变起点） |
| #FF6B00 | 橙色统计卡（渐变终点） |
| #1664FF | 主色蓝/链接 |

## 三、布局规范

```
┌─────────────────────────────────────────────┐  ← 顶部导航 56px #1A1A1A
│ Logo  工作台  商品管理  订单管理   退出    │
├──────────┬──────────────────────────────────┤
│          │                                  │
│ 侧边菜单  │  内容区 #F5F5F5                 │
│ 200px    │  ┌──────────────────────────┐  │
│ #1A1A1A  │  │  统计卡片（橙色渐变）      │  │
│          │  └──────────────────────────┘  │
│          │  ┌──────────────────────────┐  │
│          │  │  筛选条件（白色卡片）      │  │
│          │  └──────────────────────────┘  │
│          │  ┌──────────────────────────┐  │
│          │  │  数据表格（白色卡片）      │  │
│          │  └──────────────────────────┘  │
└──────────┴──────────────────────────────────┘
```

| 元素 | 规范 |
|------|------|
| 顶部导航高度 | 56px |
| 顶部导航背景 | #1A1A1A |
| 侧边菜单宽度 | 200px |
| 侧边菜单背景 | #1A1A1A |
| 内容区背景 | #F5F5F5 |
| 卡片圆角 | 8px |
| 页面内边距 | 20px |
| 卡片间距 | 12px |

## 四、顶部导航规范

| 元素 | 规范 |
|------|------|
| 高度 | 56px |
| 背景 | #1A1A1A |
| Logo文字 | 白色，字号 16px，字重 600 |
| Logo左侧 | 可能有图标 |
| 菜单文字（未选中） | rgba(255,255,255,0.65)，字号 14px |
| 菜单文字（选中） | #FFFFFF，字重 500 |
| 右侧用户区 | 通知图标 + 用户头像(32px圆形) + 用户名 + 退出 |

## 五、侧边菜单规范

| 元素 | 规范 |
|------|------|
| 背景色 | #1A1A1A |
| 一级菜单项高度 | 48px |
| 一级菜单文字 | #FFFFFF，字号 14px，字重 500 |
| 菜单图标 | 16×16px，白色 |
| 选中态 | 左侧 3px #1664FF 边框，文字 #1664FF |
| 选中态背景 | rgba(22,100,255,0.15) |
| Hover态 | 背景 rgba(255,255,255,0.05) |

**菜单结构：**
```
工作台
商品管理
　└ 商品列表 / 添加商品 / 库存管理 / SKU管理
订单管理
　└ 订单列表 / 退货单
供应商管理
　└ 供应商列表 / 我的报价
消息中心
数据统计
　└ 销售统计 / 订单分析 / 转化漏斗
系统管理
　└ 账号设置 / 操作日志
```
