# 平台端 UI 规范 Skill（ui-spec-platform）

## 使用场景
当用户提到"平台端UI"、"平台运营后台UI"、"运营后台UI"时调用。

## 规范说明

平台端与商家端、供应商端共用同一套蓝色主题规范。

**直接引用 ui-spec-merchant.md 的全部内容。**

---

## 颜色规范（蓝色主题）

```css
/* 主色 */
--color-primary: #1664FF;
--color-primary-hover: #1458DB;

/* 主按钮 */
.ant-btn-primary {
  background: #1664FF !important;
  border-color: #1664FF !important;
}

/* 顶部导航 / 侧边菜单背景 */
background: #1A1A1A;
```

## 布局规范

| 元素 | 规范 |
|------|------|
| 顶部导航 | 高 56px，背景 #1A1A1A |
| 侧边菜单 | 宽 200px，背景 #1A1A1A |
| 内容区背景 | #F5F5F5 |
| 表格行高 | 48px |

## 字体规范

| 层级 | 字号 | 字重 |
|------|------|------|
| 页面标题 | 18px | 600 |
| 区块标题 | 16px | 600 |
| 表格表头 | 13px | 600 |
| 表格内容 | 13px | 400 |
| 正文 | 14px | 400 |
| 辅助文字 | 12px | 400 |

## 圆角规范

| 元素 | 圆角 |
|------|------|
| 按钮 | 4px |
| 输入框 | 4px |
| 卡片 | 8px |
| 弹窗 | 8px |
