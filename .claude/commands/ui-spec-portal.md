# 门户商城端 UI 规范 Skill（ui-spec-portal）

## 使用场景
当用户提到"门户端UI"、"官网UI"、"商城前端UI"、"PC商城UI"时调用。

---

## 一、技术栈

| 技术 | 说明 |
|------|------|
| 框架 | Vue 3 + Vite（根据实际技术选型） |
| 样式 | SCSS |
| 主题色 | #3370FF（蓝色） |
| 辅助色 | #FF7700（橙色，用于金刚区图标背景） |
| 响应式 | 1920px / 1440px / 1280px / 1024px |

---

## 二、颜色规范

### 品牌色

| 色值 | 用途 |
|------|------|
| #3370FF | 主色蓝（按钮、链接、选中态） |
| #2860E0 | 深蓝（按钮 hover） |
| #E8F0FF | 浅蓝（标签背景、选中态背景） |
| #FF7700 | 橙色（促销、金刚区图标背景） |
| #E65C00 | 深橙（橙色 hover） |

### 功能色

| 色值 | 用途 |
|------|------|
| #F5222D | 价格红 / 错误 |
| #FF6B35 | 促销橙 |
| #52C41A | 成功绿 |
| #FAAD14 | 警告黄 |

### 中性色

| 色值 | 用途 |
|------|------|
| #333333 | 标题黑 |
| #666666 | 正文灰 |
| #999999 | 辅助灰 |
| #E5E5E5 | 分割线 |
| #F5F5F5 | 页面背景灰 |
| #FFFFFF | 卡片/页面背景 |

## 三、字体规范

| 层级 | 字号 | 字重 | 使用场景 |
|------|------|------|---------|
| 页面大标题 | 20px | 700 | 首页主标题、买家中心标题 |
| 区块标题 | 16px | 600 | 卡片标题、买家中心数据卡 |
| 正文 | 14px | 400 | 商品名称、描述、按钮文字 |
| 辅助文字 | 12px | 400 | 价格划线、标签、时间戳 |
| 最小文字 | 10-11px | 400 | 角标 |
