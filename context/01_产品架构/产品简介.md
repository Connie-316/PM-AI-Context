# 产品简介

## 产品名称
启泰数智聚模 - 物料商城集采平台

## 产品定位
模具行业 B2B 垂直电商平台，连接供应商、商家、平台三方，实现模具物料的在线采购与交易。

## 核心功能
- 供应商入驻、商品发布、在线询价/报价
- 商家采购、订单管理、供应链协同
- 平台运营、商家管理、供应商管理

## 用户角色

| 角色 | 说明 |
|------|------|
| 供应商 | 提供模具物料的商家，在平台开店 |
| 商家 | 采购方，在平台购买模具物料 |
| 平台运营 | 管理供应商、商家、商品、订单 |
| 访客 | 未登录用户，浏览官网/小程序 |

## 端说明

| 端 | 定位 |
|----|------|
| 门户端 | 平台运营管理后台（平台运营人员用） |
| 商家端 | 商家管理后台（采购方用） |
| 供应商端 | 供应商管理后台（供应方用） |
| 官网 | 对外展示网站 |
| 小程序 | 移动端采购入口 |
| H5（规划中） | 移动端轻量化入口 |
| 公众号（规划中） | 内容推送、客服入口 |

## 所属行业
模具制造 / B2B 电商 / 工业物料集采

## 商品类目

| 类目 | 说明 |
|------|------|
| 标准件 | 按国标生产的通用零部件（螺丝、轴承、螺母等） |
| 铸件 | 通过铸造工艺生产的金属零件 |
| 非标件 | 按客户需求定制的非标准零部件 |
| 刀具 | 金属加工用切削刀具（铣刀、钻头、螺纹刀等） |
| 塑料原料 | 用于注塑成型的塑料颗粒/粉料 |
| 其他 | 平台上还有很多商品尚未上架 |

**平台现状：** 功能尚不完善，部分业务逻辑未打通，目前主要靠业务员在线下跟进，客户不愿迁移到线上。

## 当前挑战

```
1. 平台功能体验差 — 业务逻辑不通，流程走不通
2. 客户不愿上线 — 习惯线下交易，不愿意把业务搬到线上
3. 商品不全 — 还有很多商品没有上架到商城
4. 核心商业模式不清晰 — PM也在探索中
```

## 账期授信流程（已可走通）

```
步骤1：线下信用评估 → 确认分数和等级
步骤2：领导分级别依次签字审批
步骤3：后台上传销售协议 + 授信评估表
步骤4：开通账期设置权限，授权金额和账期期限
```
