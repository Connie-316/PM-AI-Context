# 产品简介

## 产品名称
启泰数智聚模 - 物料商城集采平台

## 产品定位
模具行业 B2B 垂直电商平台，连接供应商、商家、平台三方，实现模具物料的在线采购与交易。

## 核心功能
- 供应商入驻、商品发布、在线询价/报价
- 商家采购、订单管理、供应链协同
- 平台运营、商家管理、供应商管理

## 用户角色

| 角色 | 说明 |
|------|------|
| 供应商 | 提供模具物料的商家，在平台开店 |
| 商家 | 采购方，在平台购买模具物料 |
| 平台运营 | 管理供应商、商家、商品、订单 |
| 访客 | 未登录用户，浏览官网/小程序 |

## 端说明

| 端 | 定位 |
|----|------|
| 门户端 | 平台运营管理后台（平台运营人员用） |
| 商家端 | 商家管理后台（采购方用） |
| 供应商端 | 供应商管理后台（供应方用） |
| 官网 | 对外展示网站 |
| 小程序 | 移动端采购入口 |
| H5（规划中） | 移动端轻量化入口 |
| 公众号（规划中） | 内容推送、客服入口 |

## 所属行业
模具制造 / B2B 电商 / 工业物料集采

| 类目 | 说明 |
|------|------|

```
母公司：中泰工业科技（创建于1999年）
├── 中国重点骨干模具企业、中国模具出口重点企业
├── 4大产业群：工装产业群、零部件产业群、启泰智能、中泰新能源

启泰智能的三大产业平台：
1. 产业数字化赋能平台（设计协同/智慧管理/人才服务/金融服务）
2. 智能设计（智一软件）
3. 产业互联网平台（数智聚模 = 你们做的物料商城）
```

## 平台定位

```
名称：数智聚模物料商城（数智聚模）
定位：模具行业的产业互联网平台
目标：成为"模具行业的京东"（对标京东自营+撮合模式混合）
聚焦：冲压、注塑、压铸、钣金和非标设备等细分行业
目标：利用数字技术重塑模具行业发展模式，让资源可交易、可共享，构建行业新生态
```

## 商业模式

```
撮合模式（类似淘宝天猫）：
- 供应商自主经营商品（价格/信息/库存）
- 平台收佣金

自营模式（类似京东自营）：
- 平台筛选标准件品类，自主生产或代工
- 购买方直接与平台交易

盈利逻辑：
平台为供应商赋能（原材料统一采购/物流/智能化）→ 成本降低 → 价格低于市场价
→ 采购方得低价 → 平台得流量和佣金
```

## 平台核心价值主张（7条）

| 编号 | 价值 | 说明 |
|------|------|------|
| 1 | 聚合供应商 | 帮助标准件供应商建立模型，与智一软件整合，一键下单销售 |
| 2 | 拓宽销售渠道 | 协助供应商销售商品 |
| 3 | 降低原材料采购成本 | 聚合企业需求，与一级供应商统一采购 |
| 4 | 降低生产成本 | 定点制造、批量化规模化、提供提效工具（智一软件/小工单系统） |
| 5 | 降低运输成本 | 整合供应商物流信息，与物流公司深度合作 |
| 6 | 提供金融授信 | 联合银行给用户做信用授信（类似京东白条/蚂蚁花呗） |
| 7 | 协助回款 | 利用供应链金融，协助企业回款 |
