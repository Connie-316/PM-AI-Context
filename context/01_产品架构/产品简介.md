# 产品简介

## 产品名称
启泰数智聚模 - 物料商城集采平台

## 产品定位
模具行业 B2B 垂直电商平台，连接供应商、商家、平台三方，实现模具物料的在线采购与交易。

## 核心功能
- 供应商入驻、商品发布、在线询价/报价
- 商家采购、订单管理、供应链协同
- 平台运营、商家管理、供应商管理

## 用户角色

| 角色 | 说明 |
|------|------|
| 供应商 | 提供模具物料的商家，在平台开店 |
| 商家 | 采购方，在平台购买模具物料 |
| 平台运营 | 管理供应商、商家、商品、订单 |
| 访客 | 未登录用户，浏览官网/小程序 |

## 端说明

| 端 | 定位 |
|----|------|
| 门户端 | 平台运营管理后台（平台运营人员用） |
| 商家端 | 商家管理后台（采购方用） |
| 供应商端 | 供应商管理后台（供应方用） |
| 官网 | 对外展示网站 |
| 小程序 | 移动端采购入口 |
| H5（规划中） | 移动端轻量化入口 |
| 公众号（规划中） | 内容推送、客服入口 |

## 所属行业
模具制造 / B2B 电商 / 工业物料集采
