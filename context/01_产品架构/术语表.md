# 术语表

## 业务术语

| 术语 | 解释 |
|------|------|
| 物料 | 模具制造所需的原材料、零部件、工具等 |
| 集采 | 集中采购，多个采购方合并订单获取更低价 |
| 询价 | 采购方向供应商询问价格 |
| 报价 | 供应商回复询价，给出价格 |
| SKU | 库存量单位，商品的最小计价单位 |
| 供应商 | 提供物料的商家 |
| 商家 | 在平台采购物料的客户（采购方） |
| 平台 | 启泰数智聚模，撮合交易的中间方 |

## 交易与结算

| 术语 | 解释 |
|------|------|
| 账期 | 买方收货后延迟付款的期限，如"月结30天" |
| 账期支付 | 基于账期垫付的支付方式，买方延期付款 |
| 账期额度 | 平台给商家核定的可使用账期支付的最大金额 |
| 授信 | 平台对商家的信用评估结果，决定账期额度 |
| 供应链金融 | 基于真实交易数据，为供应商提供融资服务（如保理、垫资） |
| 分账 | 交易完成后，货款按约定规则分给各方（如平台抽佣后给供应商） |

## 合同与签章

| 术语 | 解释 |
|------|------|
| 电子签章 | 基于 PKI 技术的电子签名/印章，具有法律效力 |
| 框架协议 | 长期合作协议，约定总体条款，单次订单按框架执行 |
| 合同 | 买卖双方签署的正式协议，包含商品、价格、交期等条款 |

## 物料分类

| 术语 | 解释 |
|------|------|
| 铸件 | 通过铸造工艺生产的金属零件 |
| 标准件 | 按国标生产的通用零部件（螺丝、轴承、导柱、导套等） |
| 非标件 | 按客户需求定制、独立按图生产的半成品 |
| 刀具 | 金属加工用切削刀具（铣刀、钻头、螺纹刀等） |
| 耗材 | 生产过程消耗品（切削油、钼丝线、钻头等） |
| 设备 | 生产加工设备（CNC、线割、深孔钻等） |
| 原材料 | 生产用钢材、吕材等 |
| 冲模 | 冲压成型模具，用于批量生产金属板材成型件 |
| 压模 | 压力成型模具，用于压铸/压制工艺 |
| 开模 | 模具开发，设计并制造新模具的过程 |
| 注塑 | 塑料成型工艺，将熔融塑料注入模具冷却成型 |
| 排产 | 安排生产计划，确定何时生产、何时交货 |
| 软件 | B2B 商城中指工业设计软件、ERP/MES 等数字化工具 |

## 商业模式术语

| 术语 | 解释 |
|------|------|
| 撮合模式 | 供应商入驻淘宝天猫式经营，平台提供销售渠道，商品自主经营 |
| 自营模式 | 平台筛选品类自主生产或代工，类似京东自营店 |
| 集中采购（集采） | 聚合企业需求与一级供应商统一采购，降低原材料成本 |
| 定点制造 | 批量化、规模化生产，降低换装和工艺编排频率 |
| 智一软件 | 启泰自有的数字化PMC系统/小工单系统，用于生产管理赋能 |
| 法大大 | 第三方电子合同/签章平台 |

## 平台状态说明

| 术语 | 解释 |
|------|------|
| 待上架 | 商品还未在商城上线，处于待发布状态 |

## 技术术语

| 术语 | 解释 |
|------|------|
| B2B | Business to Business，企业对企业电商 |
| SAAS | 软件即服务 |
