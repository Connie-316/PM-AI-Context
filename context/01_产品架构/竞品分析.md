## 二、重点竞品深度分析

### 2.1 嘉立创（JLC）— 最强参考

**基本信息**
- 成立：2010年，深圳
- 融资：D轮（IDG/红杉）
- 员工：5000+
- 业务：PCB打样→元器件商城→SMT贴片→电子全产业链

**商业模式**

```
核心逻辑："PCB打样" 是流量入口，元器件销售是利润来源

用户路径：
访问JLC官网 → PCB下单（低价/快速）→ 发现元器件需要采购 →
引导到JLC元器件商城 → 一站式购买PCB+元器件 → 打包发运

盈利结构：
- PCB：成本透明，加价率低（获客用）
- 元器件：标准价，利润率30-50%
- SMT贴片：高毛利服务
- 加工费：菲林/钢网/工程费
```

**关键指标**

| 指标 | 数据 |
|------|------|
| 日均订单 | 1万+ |
| 交期承诺 | 24小时/48小时/72小时 |
| SKU数量 | 100万+ |
| 用户复购率 | ~60% |

**启泰可以学的**

| JLC做法 | 启泰可借鉴点 |
|--------|------------|
| 低价PCB引流 → 元器件盈利 | 低价模具样品引流 → 材料/辅料盈利 |
| 交期承诺可视化（实时进度） | 订单交期透明化 |
| 计价器（输入参数→立即出价格） | 模具估价器（输入要求→出参考价） |
| 审核自动化（上传Gerber→系统解析） | 询价单自动派发供应商 |
| 客服实时响应 | 需求咨询实时接待 |

**JLC的不足（我们的机会）**
- 只做电子制造，不做模具/机械
- 侧重小批量打样，大批量仍是传统渠道
- 非标定制能力弱

### 2.2 怡合达（YHD）

**基本信息**
- 成立：2006年，东莞
- 上市：科创板（688032）
- 业务：FA工厂自动化零部件（轴承/导轨/气缸/电机等）

**商业模式**

```
核心逻辑：SKU极多+长尾需求+快速发货

产品结构（SKU超300万）：
├── 传动件（轴承/导轨/丝杠）→ 标准品，库存深
├── 气动元件（气缸/电磁阀）→ 标准品，库存深
├── 工业自动化设备 → 非标，按需定制
└── 电气元件（继电器/PLC）→ 标准品，库存深

盈利模式：
- 经销赚差价（大部分品牌是代理）
- 自产件（部分精密件自产，毛利率高）
- VMI仓（供应商寄存销售分成）
```

**关键指标**

| 指标 | 数据 |
|------|------|
| SKU | 300万+ |
| 现货率 | 95%+ |
| 平均交付周期 | 1-2天 |
| 客户数量 | 3万+ |

**启泰可以学的**

| YHD做法 | 启泰可借鉴点 |
|--------|------------|
| 超多SKU满足长尾需求 | 模具物料SKU要足够多 |
| 供应商寄售（VMI） | 供应商先供货后结款 |
| 分类清晰（行业/品牌/类型） | 物料分类体系要完善 |
| 快速发货能力 | 交期承诺要可视化 |

### 2.3 云汉芯城（ICSO）

**基本信息**
- 成立：2008年，深圳
- 业务：电子元器件小批量在线销售

**商业模式**

```
核心逻辑：正品保障 + 小批量灵活 + 技术服务

特点：
- 只卖正品（授权渠道），解决假货痛点
- 小批量起售（1片也卖），满足研发需求
- 技术支持（替代选型/BOM配单）
- 保税仓（进口元器件快速通关）

用户画像：
- 研发工程师（买几片样品）
- 小型工厂（补料/小批量）
- 维修店（单件采购）
```

**启泰可以学的**

| ICSO做法 | 启泰可借鉴点 |
|--------|------------|
| 正品保障体系 | 模具材料质量认证 |
| 小批量销售 | 支持小批量下单 |
| BOM配单服务 | 模具材料打包采购方案 |
| 技术选型支持 | 材料选型技术支持 |

### 2.4 燕秀商城（YXSC）— 最直接竞品

**基本信息**
- 成立：2015年，东莞
- 业务：模具配件/工具/材料在线销售

**商业模式**

```
核心逻辑：模具厂一站式采购平台

产品品类：
├── 模具配件（顶针/导柱/弹簧/浇口等）
├── 模具工具（量具/刀具/夹具）
├── 模具材料（钢材/铜材/铝材）
├── 标准件（螺丝/螺母/垫圈）
└── 消耗品（切削液/润滑油）

用户画像：模具厂采购/仓库管理员
```

**启泰可以对比的**

| 维度 | 燕秀 | 启泰（目标） |
|------|------|------------|
| SKU 数量 | 约50万 | 目标100万+ |
| 交易模式 | 现货销售 | 现货+询价+集采 |
| 定价 | 固定价 | 账期+授信 |
| 物流 | 自建+第三方 | 第三方 |
| 金融服务 | 无 | 有（账期/授信） |
| 目标市场 | 华南模具厂 | 全国模具产业链 |

**燕秀的不足（启泰机会）**

- 没有金融服务（不能账期）
- 没有集采模式（无法拼单）
- 没有供应商入驻（自营为主）
- 没有供应链金融

- 业务：工业品B2B综合平台

**商业模式**


|--------|------------|
| 京东仓储物流 | 合作仓储物流 |
| 企业授信体系 | 商家授信/账期管理 |
| 商品品质保障 | 供应商资质审核 |
| 售后服务体系 | 退换货/投诉处理 |

## 四、启泰数智聚模的差异化定位

### 4.1 市场机会

```
为什么模具B2B有机会？
1. 市场规模大（模具被称为"工业之母"，年采购额万亿级）
2. 现有玩家分散（燕秀50亿体量，还算小）
3. 账期需求强（模具厂赊账严重，京东/阿里解决不了）
4. 供应链复杂（模具物料种类多，非标多，平台价值大）
```

### 4.2 启泰的差异化路径

| 维度 | 嘉立创策略 | 怡合达策略 | 启泰策略 |
|------|----------|----------|---------|
| 入口 | PCB打样 | FA零件 | 模具物料+集采 |
| 盈利 | 元器件销售 | 差价+VMI | 账期+加工费+金融 |
| 壁垒 | 交期+正品 | SKU+现货 | 账期+供应链+集群 |
| 目标 | 电子制造 | 自动化设备 | 模具产业链 |

### 4.3 启泰核心价值主张

```
"模具行业的一站式物料采购平台"

核心差异化：
1. 账期授信（解决模具厂赊账痛点）
2. 集采拼单（多厂拼单降成本）
3. 供应链金融（基于交易数据融资）
4. 产业集群覆盖（模具厂聚集地本地化服务）
```

## 七、SWOT 分析

### 7.1 启泰优势（Strengths）

```
S1. 母公司中泰工业科技背景强大（1999年成立，3000+人，模具出口重点企业）
S2. 三大产业平台联动（设计协同+智一软件+数智聚模生态）
S3. 自有数字化工具（智一软件/PMC系统/小工单系统）赋能供应商
S4. 账期授信体系已可走通（联合银行授信，类京东白条模式）
S5. 严格管理+执行力强（周会/培训/加班文化）
```

### 7.2 启泰劣势（Weaknesses）

```
W1. 新平台，供应商和买家数量不足
W2. SKU不够丰富（大量商品尚未上架）
W3. 品牌认知度低（vs嘉立创/燕秀）
W4. 产品体验差，业务逻辑不通，流程走不通
W5. 客户习惯线下交易，不愿意迁移到线上
W6. 核心差异化未建立（功能做得很烂，账期授信等未跑通）
```

### 7.3 启泰机会（Opportunities）

```
O1. 模具行业B2B渗透率低，线上空间大（但客户不愿迁移是现实障碍）
O2. 账期需求强，现有平台无法满足（但功能尚未打通）
O3. 产业集群效应（模具厂扎堆，容易拉新）
O4. 政策支持（工业互联网/制造业升级）
O5. 平台功能优化空间大，改善后可直接对标竞品
```

### 7.4 启泰威胁（Threats）

```
T1. 平台体验差 → 客户不愿使用，流失率高
T2. 嘉立创/燕秀可能切入模具物料
T3. 阿里巴巴/京东工业品降维打击
T4. 供应商不愿意入驻新平台（因为平台走不通逻辑）
T5. 账期风险（坏账/逾期）如果功能打通后控制不住
```

## 八、结论与建议

### 8.1 核心结论

```
1. 模具B2B市场存在真实痛点（账期/非标），但客户不愿迁移
2. 平台功能体验差是当前最大问题（逻辑不通，没人愿意用）
3. 账期授信功能已可走通（需线下评估+领导签字+后台开通），是实际差异化
4. "集采拼单"是竞品分析时的假设，尚未确认是否真实功能
5. 当前主要是业务员在用，不是真正的终端客户
6. 核心商业模式尚不清晰，需要PM主动探索和研究
```

### 8.2 对产品规划的启示

| 竞品启示 | 对启泰PRD的指导 |
|---------|----------------|
| 嘉立创的计价器 | 做模具估价工具（输入参数→出参考价） |
| 嘉立创的正品保障 | 供应商资质审核机制 |
| 怡合达的SKU策略 | SKU分类要细，搜索要好 |
| 怡合达的VMI | 供应商寄售模式设计 |
| 燕秀的工具属性 | 工具/辅料品类要全 |
| 京东的账期 | 账期支付要做完整（授信/额度/还款） |
| **先做通基础流程** | **功能走不通的情况下，上任何高级功能都没有意义** |
| **账期授信已可走通** | **这是真实差异化，应作为核心卖点推广** |
