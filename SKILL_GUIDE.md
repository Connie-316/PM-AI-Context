# Skill 使用指南（精简版）

> 这个文件是给你看的，不是给AI看的。
> AI已经知道所有Skill，你只需要说触发词就行。

---

## 常用技能（必须记住的4个）

### 1. 写PRD → 说"快速PRD"

```
你：帮我写个[功能名]的PRD
AI：自动调用 prd-write，写完后自动保存版本+审查+推送
```

### 2. 写完PRD后给研发 → 说"输出研发文档"

```
你：帮我输出[功能名]的研发文档
AI：自动生成接口文档+研发需求确认单
```

### 3. 同步TAPD迭代版本 → 说"TAPD对齐"

```
你：TAPD对齐
AI：问你这周有什么变化，你回答，它更新迭代规划对照表
```

### 4. 规划下月任务 → 说"月度规划"

```
你：月度规划
AI：问当前月份，生成下月15号/31号关键任务规划表
```

## 怎么用得好

### 1. 先把第一个PRD写出来

```
你说："快速PRD：帮我写一个供应商入驻的PRD"
AI写出后 → 你修改 → AI自动保存+审查+推送
```

### 2. 每次写完PRD就输出研发文档

```
PRD写完 → 说"输出研发文档" → 研发签字 → 开发
```

### 3. 每周一同步TAPD

```
打开TAPD → 说"TAPD对齐" → 告诉AI这周有什么变化 → AI更新表格
```

### 4. 每月25号规划下月

```
说"月度规划" → AI问你这月和下月 → 你回答 → AI生成规划表
```
